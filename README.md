# Operator Precedence Parser Visualizer

## Project Overview
This project implements an operator precedence parser visualizer for arithmetic expressions. It demonstrates the shift-reduce parsing mechanism used in compiler design, providing step-by-step visualization of stack operations.

## Features
- Step-by-step visualization of parsing process
- Stack and input buffer display at each step
- Shift and reduce operation demonstration
- Expression syntax validation
- Precedence table viewer
- Multiple parsing modes (verbose/quick)
- Sample expressions for learning
- Error detection and reporting

## Supported Operators
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Parentheses for grouping
- Identifiers (variables a-z, numbers 0-9)
